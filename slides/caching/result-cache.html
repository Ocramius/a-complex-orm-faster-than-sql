Result Cache

Caches query results:


$usersCount = $entityManager->createQuery(
    <<<'DQL'
SELECT COUNT(u) FROM User u
'DQL'
    )
    ->setResultCacheDriver($redisCache)
    ->useResultCache(true)
    ->setResultCacheLifeTime(7200)
    ->getResult();

Cache ID depends on query string and parameters!
